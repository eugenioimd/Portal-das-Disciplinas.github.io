Array com os atributos que são atribuíveis em massa pelo método Collaborator::create.
name: Nome do colaborador.
bond: Vinculo do colaborador com o projeto.
role: Função do colaborador no projeto.
email: E-mail do colaborador.
lattes: Link da url do perfil lattes do colaborador.
github: Link da url do perfil do github do colaborador.
urlPhoto: Local da photo do colaborador no servidor.
isManager: Determina se o colaborador é coordenador ou não.
active: Determina se o colaborador está ativo no projeto.
